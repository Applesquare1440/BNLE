6.1 Create systemd service (edit username there!!!!)
cd ~/Documents/tracking_project
sudo nano /etc/systemd/system/tracking.service

6.2 Enable service

sudo systemctl start tracking.service

6.3 Check status
sudo systemctl status tracking.service

6.4 Logs checking:
